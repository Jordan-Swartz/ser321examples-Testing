You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=3&num2=4 -- multiplies two numbers
  • /github?query=users/amehlhase316/repos -- lists public GitHub repos
  • /greet?name=Jordan&lang=en -- returns a personalized greeting
  • /repeat?text=Hi×=3 -- repeats the given text

New Custom Requests:

  • /greet?name=Jordan&lang=en → returns a personalized greeting.
    Examples:
    - /greet?name=Jordan&lang=es → Hola, Jordan!
    - /greet?name=Alex&lang=fr → Bonjour, Alex!

  • /repeat?text=Hi×=3 → repeats the given text that number of times.
    Examples:
    - /repeat?text=Go×=3 → Go Go Go
    - /repeat?text=Yes×=2 → Yes Yes
